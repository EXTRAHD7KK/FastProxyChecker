Welcome to FastProxyChecker!

FastProxyChecker is a php tool designed to help the task of checking proxies, with maximum accuracy and performance. The tool supports all kinds of proxies, multi-threading, using files, and many other features.

How do I use it, and where do I get the code?

The best way to get the stable Version 1.2 is to download it straight from GitHub Pages (Download button above). The docs are available here. Currently the best version to use is Version 1.2 which has new features, and holds strong against bugs.
